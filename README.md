# Paladi Shanthan Kumar — Portfolio

My personal portfolio website — a single-page site showcasing my work as a **Data Engineer** specialising in scalable ETL pipelines on the Microsoft Azure ecosystem.

### 🌐 Live Site

**👉 [https://shanthan-paladi.github.io/](https://shanthan-paladi.github.io/)**

---

## About

An editorial, fully responsive portfolio built as a single self-contained HTML file — no frameworks, no build step. Features a light/dark theme, smooth scroll-reveal animations, and sections covering my experience, skills, projects, and impact.

## Built With

- HTML5 / CSS3 (custom design system, CSS variables)
- Vanilla JavaScript (theme toggle, mobile menu, IntersectionObserver animations)
- Google Fonts — Instrument Serif, Archivo, JetBrains Mono
- Hosted on **GitHub Pages**

## Run Locally

```bash
git clone https://github.com/shanthan-paladi/shanthan-paladi.github.io.git
cd shanthan-paladi.github.io
# open index.html in any browser
```
